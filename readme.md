## 特性

1. 和 ChatLuna 集成，直接使用 ChatLuna 的模型。
2. 支持自定义配置模型和预设

## 部署

在 Koishi 插件市场搜索 `chatluna-character`，安装后启用即可。

**插件依赖 ChatLuna，请确保在 ChatLuna 启动后再启用本插件。**
